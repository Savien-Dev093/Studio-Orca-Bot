# 🤖 Studio Orca Bot

Sistema profissional de atendimento automatizado desenvolvido para empresas que utilizam Telegram como canal de comunicação.

![Logo](assets/logo.png)

---

## 📋 Sobre

O Studio Orca Bot é uma solução desenvolvida para automatizar o atendimento ao cliente, oferecendo acesso rápido a informações da empresa, catálogo de serviços, abertura de tickets e comunicação direta com a equipe de suporte.

---

## ✨ Funcionalidades

- 📄 Informações da empresa
- 💰 Catálogo de preços
- 🎫 Sistema de tickets
- 👨‍💻 Atendimento humano
- 🗄 Armazenamento em banco de dados
- 🤖 Fluxo automatizado

---

## 🛠 Tecnologias

- Python
- Telegram Bot API
- SQLite
- python-telegram-bot
